🔧 Fix Sidebar Test

🎯 Purpose

This test will help us manually fix the sidebar state since the global.js functions aren't working.

❌ Problem Identified

The forceSidebarState function is not defined, which means global.js isn't loading properly due to syntax errors.

🔧 Manual Fix Commands

Copy and paste these commands into the browser console:

// Manual sidebar fix
const sidebar = document.getElementById('sidebar');
if (sidebar) {
sidebar.classList.add('collapsed');
localStorage.setItem('sidebarCollapsed', 'true');
console.log('✅ Manually collapsed sidebar');
}

// Check if it worked
console.log('Sidebar collapsed:', sidebar.classList.contains('collapsed'));
console.log('localStorage:', localStorage.getItem('sidebarCollapsed'));
