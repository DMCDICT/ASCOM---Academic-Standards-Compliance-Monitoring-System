🧪 Manual State Tests

Test these scenarios manually and observe the results:

Test Scenario 1: Collapse on Other Pages

  1. Go to User Account Management
  2. Click hamburger menu to collapse sidebar
  3. Check localStorage: localStorage.getItem('sidebarCollapsed')
  4. Navigate to Dashboard
  5. Observe: Does sidebar stay collapsed?

Test Scenario 2: Expand on Other Pages

  1. Go to School Calendar
  2. Click hamburger menu to expand sidebar
  3. Check localStorage: localStorage.getItem('sidebarCollapsed')
  4. Navigate to Dashboard
  5. Observe: Does sidebar stay expanded?

Test Scenario 3: Dashboard Direct Test

  1. Go directly to Dashboard
  2. Check initial state: document.getElementById('sidebar').classList.contains('collapsed')
  3. Click hamburger menu to toggle
  4. Check localStorage: localStorage.getItem('sidebarCollapsed')
  5. Navigate away and back
  6. Observe: Does state persist?

📋 Quick Debug Commands

Run these in browser console:

// Check current sidebar state
localStorage.getItem('sidebarCollapsed')

// Check if sidebar is collapsed
document.getElementById('sidebar').classList.contains('collapsed')

// Check content wrapper margin
document.querySelector('.content-wrapper').style.marginLeft

// Force collapsed state
forceSidebarState('collapsed')

// Force expanded state
forceSidebarState('expanded')

// Check if functions exist
typeof initializeSidebarState
typeof toggleSidebar
typeof forceSidebarState
