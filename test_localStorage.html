🔍 localStorage Test

🎯 Purpose

This test will help us check if localStorage is working properly and if the sidebar state is being saved correctly.

📋 Test Steps:

  1. Check Current State: Click "Check localStorage" button below
  2. Go to User Account Management: Click the link below
  3. Collapse Sidebar: Click hamburger menu to collapse
  4. Check localStorage: Come back and click "Check localStorage" again
  5. Go to Dashboard: Click Dashboard link
  6. Observe: Does sidebar stay collapsed?

🔧 Test Controls:

🎯 Test Pages:

User Account Management Dashboard School Calendar Settings
Debug output will appear here...

📋 Console Commands:

You can also run these directly in browser console:

// Check current localStorage value
localStorage.getItem('sidebarCollapsed')

// Set collapsed state
localStorage.setItem('sidebarCollapsed', 'true')

// Set expanded state
localStorage.setItem('sidebarCollapsed', 'false')

// Clear localStorage
localStorage.removeItem('sidebarCollapsed')

// Check all localStorage items
console.log(localStorage)
